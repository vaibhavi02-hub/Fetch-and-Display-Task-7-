# 🌐 Fetch Users App (Async/Await + Dark Mode)

A simple web application that fetches and displays user data from a public API using **JavaScript Fetch API with Async/Await**.  
It also includes a **Dark/Light Mode toggle with persistent theme storage**.

---
Live link: https://vaibhavi02-hub.github.io/Fetch-and-Display-Task-7-/
---

## 🚀 Features

- ✅ Fetch data using Async/Await
- ✅ Error handling with try/catch
- ✅ Display user name, email, and city
- ✅ Reload button to refetch data
- ✅ Dark/Light mode toggle
- ✅ Theme saved using localStorage
- ✅ Hover card animation
- ✅ Clean and responsive UI

---

## 🛠 Technologies Used

- HTML5
- CSS3 (CSS Variables for Dark Mode)
- JavaScript (ES6+)
- Fetch API
- Async/Await
- LocalStorage

---

Public API:
